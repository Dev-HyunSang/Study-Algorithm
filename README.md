# Study Algorithm 
> ***알고리즘은 프로그래밍을 하는 데 있어서 굉장히 중요한 요소로 작용한다.   왜냐하면 어떤 알고리즘을 사용하느냐에 따라서 굉장히 효율적인 프로그램을 개발할 수도 있고, 비효율적이고 느린 프로그램을 개발할 수도 있기 때문이다.***
